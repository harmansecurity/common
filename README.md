# windows平台上的串口调试助手

　　这个小工具还是大二上学期学习C#的时候写的，当时为了方便调试智能车使用的。网上有很多成熟的而且特别好用的串口调试助手，不过唯一的缺点就是没法自己添加功能。当时写的也就是简单功能，不过可以提供给大家当做入门学习用，另外推荐一本书《Visual C#.NET串口通信及测控应用典型实例》，主要讲了串口通信还有很多实例。
